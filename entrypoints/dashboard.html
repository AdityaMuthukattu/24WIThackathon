Welcome to Snippet! The Educational Tool

Introduction

Hi, I'm Snippet! I'm here to help you and your students explore and understand the structure of websites through interactive learning. Here’s how you can use our tool:

1. Get Comfortable with Website Structure

You can introduce your students to the basic structure of websites. By using something they are already familiar with, such as existing websites, you can seamlessly integrate coding concepts.

2. Learn About Different Components

Students can access the app through a side panel next to the website. This panel provides explanations of different components as they hover and click through them, making it easier to learn about HTML and CSS in real-time.

Example:

<div class="container">
  <h1>Hello, World!</h1>
  <p>This is a simple example of HTML code.</p>
</div>

3. Explore and Edit HTML

Students can delete certain elements of a website or customize the colors of those elements using HTML and CSS. This hands-on approach makes learning more interactive and engaging.

4. Ask Questions

If students don’t understand something, they can ask their questions to Snippet. It will do its best to answer your questions accurately and efficiently. This ensures they get the help they need, tailored to what they are working on.

5. Hover to Understand Code

When users hover over parts of the website, the side panel will display the HTML code for that element. Using a picker, they can get an explanation of the code, which helps demystify how websites are built.

6. Manipulate the DOM

You can encourage students to learn by manipulating the DOM (Document Object Model). This allows them to see the immediate effects of their changes and understand how websites respond to code.
